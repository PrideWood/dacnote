#### 06 手动安装插件

了解 OB 插件构架
[你的库名]/.obsidian/plugins，插件只会安装在库里
显示隐藏文件快捷键
Mac: `Shift+cmd+.`
![image.png|100](https://pictures-1323793543.cos.ap-nanjing.myqcloud.com/pics/20240113220802.png)

Windows: 菜单栏【查看】- 勾选【隐藏的项目】
去 Github 安装只需要安装 `.js`、`.json`、`.css` 文件，不需要下载 source code
